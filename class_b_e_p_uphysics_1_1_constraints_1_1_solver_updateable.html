◆ IsActive

bool BEPUphysics.Constraints.SolverUpdateable.IsActive
getset

Gets or sets whether or not this solver updateable is active.

When set to false, this solver updateable will be idle and its isActiveInSolver field will always be false.

When set to true, the solver updateable will run normally and update if the type's activity conditions allow it.
