Vertices Engine - Steam Platform Support

Vertices Engine is a solo developed game engine written by @rtroe under @VirtexEdgeDesign. It is writter in C# and uses @MonoGame as it's underlying foundation allowing it to run cross platform on PC, OSX, Linux, iOS and Android.
