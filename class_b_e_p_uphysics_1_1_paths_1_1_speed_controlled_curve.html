Detailed Description

Wrapper that controls the speed at which a curve is traversed.

Even if a curve is evaluated at linearly increasing positions, the distance between consecutive values can be different. This has the effect of a curve-following object having variable velocity.

To counteract the variable velocity, this wrapper samples the curve and produces a reparameterized, distance-based curve. Changing the evaluated curve position will linearly change the value.
