◆ Softness

float BEPUphysics.Settings.CollisionResponseSettings.Softness = .001f
static

Softness multiplier used by collision penetration constraints. Higher softness values allow more velocity error and make things look 'squishier'. Defaults to 0.001.

Note that this value is not used directly by constraints; it is first scaled by the raw inverse effective mass. This allows consistent behavior across objects with different masses.
