Detailed Description

This is a little experimental project designed to iteratively converge to a decent solution to full body inverse kinematics subject to a variety of constraints.

It's currently separated from the rest of BEPUphysics library internals because the immediate goal is to test out features to be potentially integrated into a blender content pipeline. BEPUphysics interactions with this system will have to go through the interfaces like everything else for now.
